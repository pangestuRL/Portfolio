Kopi Ayah UI Design with Figma

In this project, I designed the user interface (UI) for a coffee shop using Figma. I prioritized user comfort, ensuring that the interface navigation is easy to understand and enhances the customer experience. With a warm color palette and design elements inspired by the ambiance of a coffee shop, I was able to convey the desired sense of comfort and relaxation in this design.

Kopi Ayah Website

This project is the implementation of the UI design I previously created into a responsive and functional website. Using HTML, CSS, and JavaScript, I developed a website for a coffee shop that not only reflects the visual design but also provides an optimal user experience. The page includes essential elements such as a product menu, contact information, and map integration to help customers find the coffee shop. With a responsive design, this website is accessible and comfortable to use on both desktop and mobile devices.
